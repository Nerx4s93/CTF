Rust постепенно захватывает Linux, поэтому не будет лишним в нём попрактиковаться.
Прикреплён: guess (elf file)

Программа требует входа.
![{FAB4FA6E-AED3-4B0B-8E3F-6CB3A53B8697}](../../z.%20Images/{FAB4FA6E-AED3-4B0B-8E3F-6CB3A53B8697}.png)

Rust выглядит специфично в декомпиляции C++.
Ввод username и password:
![Pasted image 20260307231551](../../z.%20Images/Pasted%20image%2020260307231551.png)

Далее лишние пробелы удаляются и длина пароля сравнивается с 15, если не равна, то пароль неверный.
![{E246D84A-A863-4FC1-9BBB-6FE4952537A9}](../../z.%20Images/{E246D84A-A863-4FC1-9BBB-6FE4952537A9}.png)

Алгоритм хэширования пароля:
![Pasted image 20260307231731](../../z.%20Images/Pasted%20image%2020260307231731.png)
v11 тут - это отдельный символ пароля, нет смысла рассматривать блок кода else, если вводить только символы ASCII.
На каждой итерации значение переменной password_cache производится xor с символом пароля. Изначально password_cache = 129

В результате хэш пароля должен быть равен 239, а имя должно быть admin.
![{0228E0A0-05B2-464E-B1A6-3B1DF85C4C1C}](../../z.%20Images/{0228E0A0-05B2-464E-B1A6-3B1DF85C4C1C}.png)
