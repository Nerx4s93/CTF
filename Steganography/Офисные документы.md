# Microsoft Word (.docx)
Современные файлы `.docx`, `.xlsx`, `.pptx` — это фактически **ZIP-архивы**. Если изменить расширение на `.zip`, можно увидеть внутреннюю структуру XML-файлов и медиа-данных.
Внутри легитимного `.docx` (через ZIP-интерфейс) можно создавать свои папки или прятать данные в метаданных, не нарушая работу документа.

# Microsoft Excel (.xlsx)
В Excel существует три уровня скрытия информации:
1. **Скрытие строк/столбцов:** Контекстное меню → **"Скрыть" (Hide)**. Обнаруживается выделением всей таблицы и командой "Показать
2. **Скрытые листы (Hidden):** Правой кнопкой по ярлыку листа → **"Скрыть"**. Список скрытых листов виден через команду "Показать".
3. **"Очень скрытые" листы (Very Hidden):** * Не отображаются в обычном интерфейсе Excel. (В LibreOffice отображаются в панеле скрытных листов).
